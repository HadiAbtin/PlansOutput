# PlansOutput

This repository contains documentation of 3 diet and exercise plans for different fitness goals.

## About This Documentation

These plans are **personalized nutrition and workout programs** generated based on **general questions** (personal details, medical conditions, preferences) and **goal-specific questions** (domain-specific questions for workout and nutrition modules).

### Structure

Each plan document follows this structure:

1. **Q&A Section** — Shows all questions and answers that led to the generation of the plan
   - General questions (basic info & preferences)
   - Goal-specific questions (domain-specific for workout & nutrition)

2. **Nutrition Plan** — Detailed daily meal plans with macronutrient breakdowns

3. **Workout Plan** — Structured training program with exercises, sets, reps, and progression

### Purpose

The purpose of this documentation is to **evaluate**:
- The **format and quality** of the generated plans
- The **quality** of the questions used to build personalized programs

### Program Flexibility

⚠️ **Important Note:** These programs are **fully customizable** by the coach. Changes can be made based on:
- Scheduled sessions with the user
- User messages/feedback to the coach regarding any aspect of the program
- Coach's professional judgment and recommendations

The plans serve as a starting point and can be adjusted to better fit individual needs, progress, and circumstances.

## Overview

### Plan 1 Details
- **Goal:** Muscle Gain
- **Nutrition:** Daily targets of 3300 kcal, 150g protein, 450g carbs, 75g fat
- **Workout:** 60-day program (9 weeks) with Upper/Lower split

### Plan 2 Details
- **Goal:** Endurance
- **Nutrition:** Daily targets of 3600 kcal, 112.5g protein, 525g carbs, 75g fat
- **Workout:** 90-day program (13 weeks) with Upper/Lower split and cardio focus

### Plan 3 Details
- **Goal:** Diabetes Management
- **Nutrition:** Daily targets of 2100 kcal, 102g protein, 225g carbs, 70g fat
- **Workout:** 90-day program (13 weeks) with Upper/Lower split and moderate intensity cardio
